# Rock Paper Scissors (Web)

## Overview
This is a Rock Paper Scissors game developed using HTML, CSS, and JavaScript. Players can compete against the computer through an interactive web interface.

## Features
- Play against the computer
- Random computer choices
- Displays game result
- Simple and responsive interface

## Technologies Used
- HTML
- CSS
- JavaScript

## How to Run

1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Start playing.
